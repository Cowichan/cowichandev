Tsara Couture

Site réalisé avec le logiciel WordPress et le plugin de Ecommerce WooCommerce

1-L'identité visuelle

Les couleurs et la typographie ont été choisies avec le client.
Le logo "artisan" devait apparaître sur la page d'accueil afin d'afficher clairement le côté artisanal de la boutique Tsara couture.

2-Développement du site

    Les principales fonctionnalités développées sont :

  • Développement d'un thème WordPress personnalisé.
  • Ajout et paramétrage du plugin WooCommerce permettant de créer une boutique en ligne avec système de paiement par PayPal et/ou autres.
  • Ajouts de plugins: formulaire de contact, carte de situation, anti-spam, Google analyticator.
  • Réferencement naturel: recherche de mots clés et nom d'images optimisées.
  • Ajout et paramétrage du plugin Table Rate Shipping by Mangohour permettant de paramétrer les méthodes de livraisons selon le poids notamment.

3-L'hébergement et le fonctionnement du site

Le site est hébergé chez OVH. Le site Tsara couture a été réalisé avec le CMS WordPress. Le tableau de bord de l'administrateur permet une gestion complète du site.
